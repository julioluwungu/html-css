Principais Formatações

Negrito / Destaque

Nós podemos colocar um termo em destaque usando a tag B (não semântica).

Mas também podemos fazer isso usando a tag STRONG (semântica e recomendado)

Itálico / Ênfase

Nós podemos colocar ênfase em um termo usando a tag I (não semãntica).

Mas também podemos fazer isso usando a tag EM (semântica e recomendado)

Texto mnarcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido / Sublinhado

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3

Texto subscrito

Para inserir coisas do tipo H2O
